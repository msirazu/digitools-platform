# 🛒 Digital Tools Buying Website

## 📌 Project Name
Digital Tools Buying Website

## 📖 Description
This is a simple e-commerce website where users can browse different digital tools and add them to a cart. Users can also remove items and proceed to checkout easily. The project is built using React and Tailwind CSS with a clean and responsive design.

## 🚀 Technologies Used
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- DaisyUI
- React-Toastify
- JSON (for product data)

## ✨ Features
1. Users can view all digital products in a clean card layout.
2. Add to cart and remove from cart functionality with toast notifications.
3. Cart shows total price and allows users to proceed to checkout.

## 📱 Responsive Design
This website is fully responsive and works on mobile, tablet, and desktop devices.
